- ## Devices
  - PicoScope 2000 Series (including internal function generator)
- ## Motor
  - A2212 13T (1000KV)
  - Known resistance: 0.1 $\Omega$

- ## Measurement procedure
  - Function generator setup
    - Frequency: 10kHz
    - Magnitude: 2V (Peak-to-peak)
  - Measuring device (motor connector)
